# MyKeyChain - Консольный менеджер паролей

## **MyKeyChain** — это безопасный консольный менеджер паролей с шифрованием, написанный на Python. Программа позволяет хранить, шифровать и управлять паролями с использованием шифра Цезаря.

## **Безопасность**
Шифрование Цезаря: Все пароли шифруются перед сохранением \
Криптографическая генерация: Используется модуль secrets для генерации стойких паролей

## **Управление данными**
Множественные аккаунты: Поддержка нескольких пользователей \
JSON-хранилище: Все данные сохраняются в формате JSON \
Автосохранение: Автоматическое сохранение после каждого изменения

## **Система категорий**
10 стандартных категорий: Соцсети, Банки, Работа и др. \
Пользовательские категории: Возможность создания собственных категорий \
Группировка паролей: Автоматическая сортировка по категориям \
Фильтрация: Просмотр паролей по выбранной категории

## **Установка**
Требования: \
Python 3.7 или выше \
Стандартные библиотеки Python (нет дополнительных зависимостей)

## **Быстрый старт**
Скачайте файл mykeychain.py \
Запустите в терминале: \
python mykeychain.py

## **Функционал**
### **Главное меню**
Добро пожаловать в менеджер паролей MyKeyChain!
1. Создать аккаунт
2. Войти
3. Выход

### **Меню пользователя** 
1. Добавить пароль
2. Изменить пароль
3. Удалить пароль
4. Сгенерировать пароль
5. Показать все пароли
6. Показать пароли по категории
7. Поиск паролей
8. Выйти из аккаунта

**ВНИМАНИЕ**: Этот проект создан для учебных целей. Для хранения критически важных паролей используйте профессиональные менеджеры паролей.
