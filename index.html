def add_password(login_name: str, users: dict): ¶

Функция добавления нового пароля для пользователя.

Функция запрашивает у пользователя название ресурса и пароль. Если пароль не введён, он автоматически генерируется. Перед сохранением пароль шифруется с использованием шифра Цезаря.

Parameters
login_name:strЛогин текущего пользователя
users:dictСловарь пользователей и их данных
def caesar_cipher(text: str, shift: int, decrypt=False) -> str: ¶

Функция шифрования/расшифровки строки с использованием шифра Цезаря.

Функция выполняет сдвиг символов входной строки по заданному набору символов. Может использоваться как для шифрования, так и для расшифровки данных в зависимости от значения параметра decrypt.

Parameters
text:strВходная строка для шифрования или расшифровки
shift:intВеличина сдвига символов
decrypt:boolФлаг расшифровки (True — расшифровка, False — шифрование)
Returns
strCтрока-результат после шифрования или расшифровки

def generate_password(length=12, use_digits=True, use_special=True): ¶

Функция генерации случайного пароля.

Функция создаёт пароль заданной длины с использованием букв латинского алфавита. По желанию пользователя в пароль могут быть включены цифры и специальные символы. Для генерации используется криптографически стойкий генератор случайных чисел.

Parameters

use_digits:boolФлаг использования цифр в пароле
use_special:boolФлаг использования специальных символов в пароле
Returns
strСгенерированный пароль
